## 二、普通用户操作流程

### 2.1 登录

1. 打开网址：http://43.157.31.73:3000
2. 输入分配的账号和密码
3. 点击「登录」按钮

> 登录失败超过20次将被锁定15分钟。

### 2.2 切换周

登录后底部有两栏 Tab：

- **上周** — 上一个自然周的文件夹
- **本周** — 当前自然周的文件夹

默认进入上周，可随时点击切换。

> Tab 标签（上周/本周）会根据当前日期自动计算，无需手动操作。

### 2.3 上传照片

1. 点击中央虚线区域（"点击选择图片"）
2. 从相册选择 JPG、PNG、GIF、WebP 图片
3. 可同时选择最多 **20 张**
4. 单张最大 **20MB**
5. 选完后点击右下角「**上传**」按钮
6. 等待进度条走完，显示成功提示

> 也可以直接拖拽图片文件到上传区域。

### 2.4 查看已上传照片

上传完成后，图片自动出现在下方的"我的照片"画廊区域。点击任意照片可全屏预览。

### 2.5 删除照片

1. 点击照片进入预览模式
2. 点击右上角「🗑」删除按钮
3. 确认删除

### 2.6 确认本周完成

当本周所有照片都上传完毕后：

1. 点击「**确认完成本周上传**」按钮
2. 确认后不可撤回
3. 等待其他人也完成上传

> **所有人完成上传后**，系统会自动打包本周所有照片，发送到管理员邮箱。

### 2.7 常见问题

| 问题 | 解决方法 |
|------|----------|
| 上传失败 | 检查网络连接，确认图片格式为 jpg/png/gif/webp，且单张 < 20MB |
| 无法登录 | 确认账号密码正确，联系管理员重置 |
| 提示"已上传完毕" | 该周已被管理员标记完成，无法继续操作 |
| 底部 Tab 显示异常 | 刷新页面即可（通常由浏览器缓存引起）|

## 三、管理员操作流程

### 3.1 登录

使用账号 `admin` / `admin123` 登录，登录后进入管理面板。

### 3.2 管理面板介绍

管理面板顶部有三个标签页：

| 标签 | 功能 |
|------|------|
| 👥 用户 | 查看所有用户、添加新用户 |
| 📁 周 | 查看所有周、手动打包、重置状态 |
| 📊 记录 | 查看所有用户的确认记录 |

### 3.3 添加用户

1. 进入「👥 用户」标签
2. 点击右上角「**+ 添加**」按钮
3. 填写：显示名称、登录账号、登录密码、角色
4. 点击「确认添加」

### 3.4 查看本周上传情况

进入「📁 周」标签，查看各周的完成状态：
- **待上传**（黄标）— 有用户尚未确认
- **已上传**（绿标）— 所有人已确认，已打包

### 3.5 手动打包发送邮件

当需要提前打包时（如有人忘确认但已上传完毕）：

1. 进入「📁 周」标签
2. 找到对应周，点击右侧「📦」按钮
3. 系统将立即打包并发送邮件

### 3.6 重置周状态

当某周需要重新上传时：

1. 进入「📁 周」标签
2. 找到对应周，点击右侧「🔄」按钮
3. 所有确认记录将被清除，该周恢复"待上传"状态

### 3.7 配置邮件通知

首次使用需配置邮件：

1. 在服务器项目目录找到 `.env.example`
2. 复制为 `.env`
3. 填写 QQ 邮箱授权码：
   ```
   QQ_EMAIL_ACCOUNT=your_email@qq.com
   QQ_EMAIL_AUTH_CODE=your_auth_code
   ```
4. 授权码获取方式：QQ邮箱网页版 → 设置 → 账户 → POP3/SMTP服务 → 生成授权码
