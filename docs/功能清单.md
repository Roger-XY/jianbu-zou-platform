# 健步走上传平台 — 功能清单

> 本文档记录所有已实现功能点及版本历史，供后续迭代更新参考。
> 当前版本：v2.1.0（2026-05-18）

---

## 一、基础信息

| 项目 | 内容 |
|------|------|
| 项目名称 | 健步走上传平台（jianbu Zou Uploader） |
| 技术栈 | Node.js + Express + SQLite + Multer |
| 数据库 | SQLite（WAL 模式，支持 better-sqlite3） |
| 部署方式 | PM2 进程管理（腾讯云 CentOS） |
| 部署地址 | http://43.157.31.73:3000 |

### 2.2 周文件夹模块

| 功能点 | 状态 | 说明 |
|--------|------|------|
| 周自动生成 | ✅ 已实现 | 服务启动时自动创建近4周+未来1周（第5周起） |
| 周命名规则 | ✅ 已实现 | 第N周 + 开始日期 + 结束日期，如"第7周0518-0524" |
| 周编号计算 | ✅ 已实现 | 以 2026-05-04（周一）为第5周基准，按自然周递增 |
| 动态周标签 | ✅ 已实现 | 底部 Tab 显示"上周/本周"，基于实时日期自动计算 |
| 普通用户周列表 | ✅ 已实现 | 仅返回上周+本周（2条），按 week_end < today 自动选中上周 |
| 管理员周列表 | ✅ 已实现 | 返回全部周 |
| 周详情查询 | ✅ 已实现 | GET /api/weeks/:id |

**API 路由：**
- `GET /api/weeks` — 获取周列表（按权限返回不同范围）
- `GET /api/weeks/:id` — 获取某周详情

**数据库表：** `weeks`
- `id, week_number, year, week_start, week_end, folder_name, status, zip_path, created_at, updated_at`
