# 测试套件

本目录包含健步走上传平台的功能测试和安全测试。

## 测试文件

| 文件 | 说明 |
|------|------|
| `functional.test.js` | 功能测试：登录、上传、权限、API接口 |
| `security.test.js` | 安全测试：认证绕过、注入、暴力破解、越权访问 |

## 运行测试

```bash
# 安装依赖（如果尚未安装测试所需包）
npm install

# 运行所有测试
npm test

# 仅运行功能测试
npm test -- functional

# 仅运行安全测试
npm test -- security
```

## 测试前提

1. 服务必须运行在 `http://localhost:3000`
2. `.env` 文件已配置（邮箱授权码可为空，不影响测试）
3. 数据库为干净的初始状态（未上传过任何文件）
