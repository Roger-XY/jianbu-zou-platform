# 测试套件 v2.1

本目录包含健步走上传平台的功能测试、安全测试和 E2E 全流程测试。

## 测试文件

| 文件 | 说明 |
|------|------|
| `functional.test.js` | 功能测试：登录、上传、权限、API接口（32个用例） |
| `security.test.js` | 安全测试：认证绕过、注入、暴力破解、越权访问（22个用例） |
| `e2e.test.js` | E2E 全流程：周标签、日期更新、上传、确认完成（12个用例） |

## 运行测试

```bash
# 确保服务已启动
node server.js &

# 运行所有测试
npm test

# 仅运行功能测试
npm run test:func

# 仅运行安全测试
npm run test:security

# 运行 E2E 全流程测试
node tests/e2e.test.js
```

## 重要注意事项

1. **服务必须运行在 `http://localhost:3000`**
2. 运行安全测试后，暴力破解测试会触发 15 分钟 rate limit，需重启服务才能清除
3. **正确测试顺序**：先运行 `functional.test.js`，再运行 `security.test.js`（因为安全测试末尾会触发 rate limit）
4. 两套测试之间需重启服务
